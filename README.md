## Getting Started<hr />



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Novianaa/SurveyApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Then, you can running this project with the command below
   ```js
   npm run android
   ```
